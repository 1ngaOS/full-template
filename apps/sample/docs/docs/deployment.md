# Deployment

## Frontend Deployment

The frontend is built as a static site and can be deployed to:

- Cloudflare Pages
- Firebase Hosting
- Azure Blob Storage
- Any static hosting service

## Backend Deployment

The Rust backend can be deployed as:

- Docker container
- Standalone binary
- Cloud service (AWS, Azure, GCP)

## Documentation Deployment

The documentation site can be deployed to:

- Cloudflare Pages
- Netlify
- Vercel
- Any static hosting service
