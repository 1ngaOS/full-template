# Introduction

Welcome to the Sample App documentation!

This is a sample application built in a monorepo structure demonstrating best practices.

## What is this app?

The Sample App is a full-stack application with:

- **Frontend**: SvelteKit with static site generation
- **Backend**: Rust API server with Axum
- **Documentation**: Docusaurus documentation site

## Features

- Modern tech stack
- Type-safe development
- Comprehensive testing
- Static site generation for optimal performance
