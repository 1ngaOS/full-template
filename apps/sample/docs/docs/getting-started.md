# Getting Started

This guide will help you get started with the Sample App.

## Prerequisites

- Node.js >= 18.0.0
- pnpm >= 8.0.0
- Rust (for backend development)

## Installation

```bash
# Install dependencies
pnpm install
```

## Development

### Frontend

```bash
pnpm sample:fr dev
```

### Backend

```bash
pnpm sample:be dev
```

### Documentation

```bash
pnpm sample:docs start
```

## Building

```bash
# Build all components
pnpm build

# Build specific component
pnpm sample:fr build
pnpm sample:be build
pnpm sample:docs build
```
