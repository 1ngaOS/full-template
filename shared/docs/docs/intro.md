# Shared Services Documentation

Welcome to the unified documentation for shared services in the monorepo.

## Overview

This documentation site provides information about all shared services and applications that are used across multiple apps in the monorepo.

## Shared Services

Shared services are located in the `shared/` directory and include:

- Global authentication services
- Gateway microservices
- Federation services
- Communication services
- Other cross-cutting concerns

## Documentation Structure

Each shared service should have its documentation integrated into this unified documentation site.
